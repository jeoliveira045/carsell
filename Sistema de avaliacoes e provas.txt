Objetivo do sistema: Servir como banco de dados de questoes para que os professores possam montar provas e questionarios de forma mais simples e facil.
Objetivo do sistema 2: Servir como um base para verificar o desempenho dos alunos

Quem sao os atores do sistema

Professor
Aluno
Administracao

O problema: Preciso criar uma apostila ou avaliacao para os alunos. Para isso, preciso levantar questoes com base na materia que estao dando em uma determinada escola, em um determinado periodo, para uma determinada turma que esta em um determinado periodo academico. Todas essas variaveis influenciam na questao da apostila. Esse tipo de levantamento requer muito esforco. Outro problema e que o professor nao consegue ter uma perspectiva de como esta o desempenho do aluno sem um lista de parametros clara, o que impede que o aluno possa ser avaliado de forma eficaz.

A solucao: Um sistema que permita que os professores possam montar sua propria base de dados de questoes, permita que questionarios e avaliacoes possam ser criadas com base nesses dados e permita que alunos possam realizar os questionarios e avaliacoes que os professores passarem. A plataforma tambem permitira que o desempenho do aluno seja metrificado com base nas questoes que ele resolver nos questinarios, o que podera suprir o professor com uma base mais solida para avaliar aquele aluno quanto a determinada materia.
